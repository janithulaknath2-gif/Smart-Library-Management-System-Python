Smart Library Management System (Python)

A menu-driven *Library Management System built using Python*, designed for beginners to understand real-world programming concepts such as authentication, data handling, loops, functions, and validations.

This project allows users to **sign up, log in, borrow books, return books, calculate late fines**, and **view their activity summary**.

Features

Secure Login & Signup System  
Strong Password Validation  
Book Borrowing System  
Book Returning System  
Late Return Fine Calculation  
User Activity Tracking  
Menu-Driven Interface  
Beginner Friendly & Well Structured Code  



Technologies Used
  - Functions
  - Lists & Dictionaries
  - Loops & Conditions
  - Input Validation & Error Handling
